# SpoonBenders

Realtime Multiplayer Game

- Communication with MQTT (CocoaMQTT)
- Programmatic UI
- UIKit
- MVC
- Animations and Sounds
- No game engine, just native swift (no SpriteKit)
- No third party library except CocoaMQTT
